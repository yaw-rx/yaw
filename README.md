# YAW

A reactive web framework built on custom elements, RxJS, and the DOM.

```bash
npm install
npm run build
```

## Packages

| Package | Description |
|---|---|
| [@yaw-rx/core](packages/yaw) | Core runtime - state, components, DI, bindings, directives, routing, hydration |
| [@yaw-rx/common](packages/yaw-common) | Shared template and style transform library |
| [@yaw-rx/transformer](packages/yaw-transformer) | TypeScript build-time transformer (ts-patch) |
| [@yaw-rx/ts-plugin](packages/yaw-ts-plugin) | TypeScript language service plugin |
| [@yaw-rx/ts-plugin-audit](packages/yaw-ts-plugin-audit) | Audit tooling for the TS plugin |
| [@yaw-rx/ssg](packages/yaw-ssg) | Static site generator (Puppeteer) |
| [@yaw-rx/vscode](packages/yaw-vscode) | VSCode extension - inline template and style highlighting |
| [@yaw-rx/vite](packages/yaw-vite) | Vite plugin |
| [@yaw-rx/rollup](packages/yaw-rollup) | Rollup plugin |
| [@yaw-rx/esbuild](packages/yaw-esbuild) | esbuild plugin |
| [@yaw-rx/webpack](packages/yaw-webpack) | Webpack loader |
| [yaw-example](packages/yaw-example) | Example application and documentation site |
